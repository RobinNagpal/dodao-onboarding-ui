## Prerequisites
Install `yarn` version - `1.22.17`
Install `node` version - `v16.14.2` 

## Project setup
copy `.env.example` and name it as `.env.local`

```
yarn
```

### Compiles and hot-reloads for development

```
yarn dev
```

### Compiles and minifies for production

```
yarn build
```

### Lints and fixes files

```
yarn run lint
```
